Local GitLab Installation with LXC containers
Aug 20, 2022

Hello, let us install GitLab (community edition), locally, with lxc container, from Debian FastTrack repository. Install lxc sudo apt install lxc Create and attach to a bullseye (unstable for testing) container sudo lxc-create -n gitlab -t debian -- -r bullseye sudo lxc-start -n gitlab sudo lxc-attach -n gitlab Install keyring and repos of fasttrack apt install fasttrack-archive-keyring ca-certificates echo 'deb https://fasttrack.debian.net/debian/ bullseye-fasttrack main contrib' > /etc/apt/sources.list.d/fasttrack.list echo 'deb https://fasttrack.debian.net/debian/ bullseye-backports-staging main contrib' >> /etc/apt/sources.
