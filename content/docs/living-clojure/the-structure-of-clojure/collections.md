The collections are necessary utils to organize and access data.

### Lists
putting quotes around text.
```clojure
"hello world"
;; -> "hello world"

```

### Vectors
Numeric value[s] as following
```clojure
47
;; -> 47
```

### Maps
Representation of values which won't turn into decimals when evaluated.
```clojure
1/9
;; -> 1/9
```

### Sets
```
3.14
;; -> 3.14
```
