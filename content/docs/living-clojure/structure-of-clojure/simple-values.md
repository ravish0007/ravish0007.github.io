The simple values of Clojure are as follows,

## Strings

putting quotes around text.

```clojure
"hello world"
;; -> "hello world"

```

## Integers

Numeric value[s] as following

```clojure
47
;; -> 47
```

## Ratios

Representation of values which wonâ€™t turn into decimals when evaluated.

```clojure
1/9
;; -> 1/9
```

## Keywords

Symbolic identifiers and starts with a colon.

```clojure
:nice
;; -> :nice
```

## Characters

preceding with backslash creates characters.

```clojure
\j
;; -> \j
```

## Booleans

```clojure
true
;; -> true

false
;; -> false

;; absense of value
nil
;; -> nil
```
